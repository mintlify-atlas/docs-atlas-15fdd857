# skyhan.cloud

skyhan.cloud is an open-source personal portfolio website built by [Gökhan Bulut](https://skyhan.cloud), a TypeScript developer and freelancer from Turkey. The site showcases projects, work history, and a technical blog — all powered by Next.js 15, MDX, and Tailwind CSS.

## What's included

skyhan.cloud is a fully featured personal portfolio site. Here's what it ships with:

- **Homepage** — Bio, inline tooltips, projects list, work history, blog posts, and a contact section
- **Blog** — MDX-based articles with syntax-highlighted code blocks, images, and copy buttons
- **Discord status** — Real-time online/offline indicator and Spotify listening status powered by [lantern.rest](https://lantern.rest)
- **Analytics** — [Rybbit](https://www.rybbit.io) analytics with custom event tracking
- **View transitions** — Smooth animated page navigation via `next-view-transitions`
- **Light/dark mode** — Automatic theming based on OS preference, using CSS variables
- **Resume** — Static PDF served at `/resume.pdf`

## Tech stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router, Turbopack) |
| Styling | Tailwind CSS 3.4 |
| Content | MDX via `@next/mdx` |
| State | Zustand |
| UI primitives | Radix UI (Tooltip) |
| Analytics | Rybbit |
| Discord API | lantern.rest |
| Icons | react-icons |
| Date formatting | date-fns |
